
### 복습
- boolean 값이 참이면 if문 아니면 else if ~ else 문 생성 - 중첩된 조건문 *들여쓰기 유의하기 
- if문 관계연산 가능
- 고정 변수는 final 상수로 고정.
- [ ] 값교환시 새로운 공간 생성 예시
- 나누기 계산시 0 오류 안내. 
- 입력값 여러개 있을 때 관계 연산 활용하기.
- 처음부터 특정 단위 입력 설명과 사용자가 실수형태로 입력하게 한다.
-  if ~ else if ~ else 전 조건이 미만일 경우 포함된 해당 범위는 안적어도 된다.
