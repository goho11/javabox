
###### 명시적 형변환
- 어떤 자료형을 다른 자료형으로 변환. 형태를 붙여야 한다
- 큰 값을 작은 값으로 변환시 일부 손실
	double pi = 3.14;
	int three = ==(int)== pi;
	3.14 > 3 (값 손실)
	연산시 값하나에 하나씩 붙여야 한다. (int)1.4 + 5.5 = 6.5

###### 자동형변환 (AutoCasting)
- 작은 값을 큰 값에 옮길 때 자동변환
- 큰 값을 작은 값으로 바꿀수 없다
	- 정수와 실수도 없다. 
	  float > long 변환 불가
